To-Do List App (Tkinter)

A simple and user-friendly To-Do List desktop application built using Python’s Tkinter library. This app allows users to manage daily tasks efficiently with a clean graphical interface.

FEATURES
* Add new tasks
* Remove selected tasks
* Delete all tasks (with confirmation)
* Warning messages for invalid actions
* Exit the application easily

USAGE
1. Type a task into the input field
2. Click "Add Task" to add it to the list
3. Select a task and click "Remove Selected Task"
4. Click "Delete All Tasks" to clear the list
5. Click "Exit" to close the app
